Rat Smash Game

MIT App Inventor

This project is a two-player mobile game where players compete to smash as many rats as possible within one minute. The player with the highest score at the end of the timer wins the match. Built using MIT App Inventor, this game combines fun, speed, and competition in a simple yet exciting interactive experience.

Home Screen

This is the first screen players will see before starting the game. Both players must enter their names in the provided fields and press the “Submit” button to continue. Once submitted, their names will be displayed throughout the game, creating a more personalized and competitive experience.

Main Menu

After submitting their names, players are directed to the Main Menu. Here, they can select the difficulty level of the game — Easy, Medium, or Hard. The harder the level, the faster the rat moves, making it more challenging to tap. The Main Menu also includes a “Leaderboard” button where players can view the highest scores achieved in previous rounds.

Instructions

Once a difficulty level is chosen, Instructions will pop-up. It briefly explains how the game works and outlines the main rules. It ensures both players understand the gameplay mechanics, such as how to earn points and what happens when the timer runs out, before they begin.

Gameplay Page

This is the main gameplay screen where the real competition begins. The screen is split into two areas, one for Player One and one for Player Two. Each player must tap the rat as quickly as possible within their side of the screen to score points. A countdown timer is displayed at the top, gradually ticking down until it reaches zero, signaling the end of the match.

LeaderBoard Page

Once the timer reaches zero, players are automatically redirected to the Leaderboard page. This screen displays the final scores of both players and determines the winner of the round. The leaderboard also keeps track of previous game results, allowing players to compare scores and aim for new high records in future matches.
