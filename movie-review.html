Movie Review Website

HTML, CSS, JavaScript, & Database Integration

This project is a dynamic movie review platform where users can register, log in, add reviews, manage a personal wishlist, and watch trailers. Built with HTML, CSS, JavaScript, and MySQL... designed to create a simple yet interactive movie experience for users.

Login or SignUp Page

User can choose to either login or sign up. If a first-time user, user should sign up or if an existing user, user should login.

Contact Us Page

If user is having trouble with login/signup, they can press the “Contact Us” to send an enquiry form to the administrator.

Login Page

As an existing user, user can login to their account by entering their email address and password. If the user does not have an account yet, they can tap on “Sign up” to create a new account. If the user has forgotten their password, the user can press on “Forgot password?” to reset their password.

User Homepage

After login, the user will be able to view the home page with a list of movies based on the genre. User can find a movie by searching for it rather than scrolling through the entire list of movies. This makes the search process easier for the user. If user wants to watch a movie in a specific genre or language, they can filter that too through the “Categories”.

Filtered Categories

Once filtered, only the selected option of movies will be shown. As shown, the option “Horror” is chosen in Categories and only Horror movies will be shown. User can choose “All genres”, “All languages” or refresh the page to view the whole list of movies again.

Newly Added Movies

User will be able to view the newly added movies under the “Newly Added” section.

Movie Synopsis Page

After tapping on a movie, user will be able to view the movie details along with its trailer. User will also be able to watch the trailer straight from the synopsis page. This makes it convenient for user to know better about the movie they’d like to watch.

Add To Watchlist/Favourites and Submit Review

On the same page as the movie details, user will also be able to add the movies to favourite or watchlist, add their review and view other users’ reviews as well.

More Recommended Like This

At the bottom of the same page, user will be able to view more recommended movies as the movie they have clicked. It is based on the movie genre of the movie user have clicked. As the image above, user have clicked on the Wicked movie which is under the fantasy genre. Hence, only movies under the fantasy genre will be shown in a movie grid.

Movie Reviews

User will be able to view other users’ reviews through “View Reviews”.

User Profile

On the “Profile” page, user will be able to update their profile picture, enter their username and bio. The replies to enquiries user have sent can be viewed on “Profile” page by tapping on the message icon.

Update User Profile

User can update their account’s information through “Settings” page.
