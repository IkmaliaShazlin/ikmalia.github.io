Retail Outlet Shopping System

C++ Interactive Shopping Experience

This project simulates a retail outlet shopping experience. Customers can input their gender, browse items by category, choose colors and sizes, add multiple items to their cart, and generate a receipt with the total payment. Designed to make shopping seamless and organized!

Gender Input

The system first asks the customer to input their correct gender code, ensuring personalized shopping options.

Category Selection

Customers can choose between "Clothings" or "Accessories". The system then displays all available items in the selected category. Users pick their desired item code to proceed.

Item Colour & Size

Customers can select their preferred item color and size before adding the product to the shopping cart.

Add More to Cart

Users can continue shopping and add more items to their cart, repeating the same process for multiple products.

Receipt Generation

Finally, the system asks for the customer's name and address and generates a receipt with the total payment, summarizing all purchased items.
